# Task Management

Web backend application for lessons and students with GraphQL and MongoDB

<br>

## Project setup

```bash
$ npm install
```

### Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

<br>

## Languages and Tools

- VSCode
- NestJs
- TypeORM
- GraphQL
- MongoDB
- Robo 3T
